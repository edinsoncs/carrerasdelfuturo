Profesional Frontend Developer

Convierte en un verdado frontend developer profesional, Aprende lo último en tecnologías web de lado del cliente. html5, jade, handlebars, css, stylus, less, sass, responsive design, mobil first, boostrap, foundation.

CON ESTE CURSO OBTIENES

  • Acceso de por vida
  • Asistencia del profesor
  • Certificado Online
  • Clases de apoyo
  • Material y servicios de apoyo

Aprende desde lo mas básico hasta lo mas avanzado. Vas a conocer todas las técnicas y tecnologías nuevas que se aplica en el mundo web, las etiquetas de HTML5, utilizaremos templates engine como JADE Y HANDLEBARS

Compilaremos nuestro JADE Y HANDLEBARS con la consola gitbash y prepros. Utilizaremos ICONFONTS Y SVG, animaciones con Css3 - Keyframes.

Aprenderás todas las técnicas de los preprocesadores mas conocidos como SASS, LESS, STYLUS y a utilizar el lenguaje de programación de lado cliente Javascript.

Utilizaremos git bash y github, crearemos nuestro primer commit y subiremos a nuestro repositorio, te enseñare a trabajar en equipo y alojaremos nuestras páginas web dentro de github.

Crearemos paginas web totalmente responsive design y mobil first

Veremos un bonus como utilizar Boostrap y Foundation

Optimización en paginas web y posicionamiento con HTML5.
